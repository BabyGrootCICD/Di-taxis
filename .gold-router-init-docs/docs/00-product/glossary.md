# Glossary

## Units
- **g (gram)**: base unit used across the app UI
- **troy ounce (ozt)**: 31.1034768 grams (used for XAUt representation)

## Tokens / platforms
- **XAUt**: gold-backed token (1 token ≈ 1 troy ounce)
- **KAU**: gold-backed unit (1 token ≈ 1 gram)
- **Meld Gold (GOLD$)**: gold-linked ASA (project-specific unit; verify issuer terms)

## Trading and risk
- **Slippage**: difference between expected and executed price
- **Depth**: available liquidity at/near a price level
- **Circuit breaker**: automatic halt under abnormal conditions

## Security
- **Least privilege**: minimal permissions for API keys
- **Audit log**: tamper-evident record of security-sensitive actions
